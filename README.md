# For runing project :

1.copy.env files base .env.example and enter dataabase info

2.git checkout release/v-1  

3.composer install

4.npm install

5.php artisan key:generate

6.php artisan migrate

7.php artisan db:seed

8.php artisan serve

9.npm run dev
